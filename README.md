# Praha syysloma 2026

Staattinen GitHub Pages -sivu kolmen Prahan majoituskohteen vertailuun.

## Julkaisu GitHub Pagesiin

1. Vie nämä tiedostot GitHub-repoon.
2. Avaa GitHubissa `Settings` -> `Pages`.
3. Valitse lähteeksi repohaara ja kansioksi `/root`.
4. Tallenna. Sivun pääosoite avaa `index.html`-tiedoston.

Sivusto ei tarvitse build-komentoa. Kartta käyttää OpenStreetMap/Leaflet-kirjastoa CDN:stä, ja kuvapaneelit latautuvat joko paikallisista `assets`-tiedostoista tai hostin julkisesta kuvasta.
